# userStories-notifications-evidence

**User Story:**  
As a user, I want to receive timely notifications about my habits, so I don't forget to complete them.

**Acceptance Criteria:**  
- User can turn notifications on/off for each habit.
- Notification times can be set/batched.
- User receives daily reminder notifications.

# userStories-detailScreen-evidence

**User Story:**  
As a user, I want to see detailed information about each habit so I can track and update my progress.

**Acceptance Criteria:**  
- Detailed view displays habit description, progress, and color.
- User can edit or delete habits from the detail screen.
- User can mark a habit as completed.

# userStories-homeScreen-evidence

**User Story:**  
As a user, I want to see an overview of my habits and progress on the home screen so I can stay motivated.

**Acceptance Criteria:**  
- Home screen displays summary and active habits.
- Shortcut to add habits from the home screen.
- Welcome message or quick guide for new users.

# userStories-settingscreen-evidence

**User Story:**  
As a user, I want to change account details and adjust application preferences in the settings screen.

**Acceptance Criteria:**  
- User can update name, email, password.
- User can switch between dark and light modes.
- Notification and privacy settings available.

# userStories-menusettings-evidence

**User Story:**  
As a user, I want to access a settings menu from anywhere in the app so I can quickly adjust my preferences.

**Acceptance Criteria:**  
- Settings menu accessible from all screens.
- Menu has clear sections (Profile, Notifications, Appearance).
- Admin can restrict access to sensitive settings.

# userStories-externalAPI-evidence

**User Story:**  
As a user, I want to view external data (e.g., weather updates or map results) integrated in the app for a richer experience.

**Acceptance Criteria:**  
- Weather API shows real-time data.
- Map API displays location-based features.
- External tips/news fetches motivational content.

# userStories-login-registration-evidence

**User Story:**  
As a user, I want to register and securely log in with my personal details.

**Acceptance Criteria:**  
- User can sign up with name, email, password.
